Simiki

Quick Start

Install

pip install simiki

Update

pip install -U simiki

Init Site

mkdir mywiki && cd mywiki
simiki init

Create a new wiki

simiki new -t "Hello Simiki" -c first-catetory

Generate

simiki generate

Preview

simiki preview

For more information, simiki -h or have a look at Simiki.org
