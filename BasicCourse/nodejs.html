NodeJs

准备安装

yum -y install gcc make gcc-c++ openssl-devel wget

准备更新

yum -y update gcc make gcc-c++ openssl-devel wget

源码安装

wget http://nodejs.org/dist/v0.10.26/node-v0.10.26.tar.gz
tar -zvxf node-v0.10.26.tar.gz

编译安装

make && make install

验证

node -v
npm -v
